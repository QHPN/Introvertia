# Social Media Application - Đồ Án Cơ Sở

Dự án Website mạng xã hội (Social Media Application)

## Công nghệ sử dụng

### Front-end:

- React 19
- Next.js 15
- TypeScript
- Tailwind CSS
- NextAuth.js
- Socket.io

### Back-end:

- Node.js
- Next.js 15 (Server Actions)
- Prisma ORM

### Database:

- MySQL

### Deployment:

- Vercel (frontend/backend)
- Railway (MySQL Cloud)

## Hướng dẫn cài đặt và triển khai

1. Clone repo
```bash
git clone https://github.com/yourusername/socialmedia-app.git](https://github.com/QHPN/introvertia.git
```

2. Cài đặt dependencies
```bash
npm install
```

3. Chạy ứng dụng local
```bash
npm run dev
```

## 🔗 Triển khai

Dự án được triển khai bằng [Vercel](https://vercel.com/) và Database sử dụng MySQL trên Railway Cloud.
